include е’Ң import зҡ„еҢәеҲ«

2017-09-05
жҠҖжңҜ
iOS

д»ҠеӨ©жҲ‘д»¬жқҘдәҶи§ЈдёӢйқўиҝҷеҮ з§ҚеҢ…еҗ«ж–Үд»¶зҡ„ж–№ејҸжңүд»Җд№Ҳзү№зӮ№е’ҢеҢәеҲ«пјҡ

дёҖгҖҒ#include

еӯҰиҝҮ C иҜӯиЁҖзҡ„дәәйғҪзҹҘйҒ“пјҢ#include е…¶е®һжҳҜдёҖдёӘйў„еӨ„зҗҶе‘Ҫд»ӨгҖӮе®ғдјҡеңЁйў„еӨ„зҗҶзҡ„ж—¶еҖҷз®ҖеҚ•зҡ„жҠҠиў« #include еҢ…еҗ«зҡ„ж–Үд»¶еҶ…е®№иҝӣиЎҢеӨҚеҲ¶зІҳиҙҙгҖӮжҲ‘д»¬жқҘзңӢзңӢдёӢйқўзҡ„д»Јз Ғпјҡ

жҲ‘д»¬дҪҝз”Ё gcc -E B.h е‘Ҫд»ӨжқҘзңӢзңӢз»ҸиҝҮйў„еӨ„зҗҶеҗҺзҡ„ж–Үд»¶еҶ…е®№еӨ§жҰӮеҰӮдёӢпјҡ

жҲ‘д»¬еҸҜд»ҘзңӢеҲ°з»ҸиҝҮйў„еӨ„зҗҶд№ӢеҗҺпјҢA.h ж–Үд»¶дёӯзҡ„еҶ…е®№иў«зӣҙжҺҘеӨҚеҲ¶е№¶зІҳиҙҙеҲ° B.h ж–Үд»¶дёӯжқҘгҖӮеҰӮжһңжҲ‘д»¬еңЁ B.h ж–Үд»¶дёӯеӨҡж¬ЎеҢ…еҗ«дәҶ A.h ж–Үд»¶пјҢдјҡеҮәзҺ°д»Җд№Ҳжғ…еҶөпјҹжҜ”еҰӮпјҡ

з»ҸиҝҮйў„еӨ„зҗҶд№ӢеҗҺзҡ„еҶ…е®№еӨ§жҰӮеҰӮдёӢпјҡ

A.h ж–Үд»¶дёӯзҡ„ sampleA() еҮҪж•°еҮәзҺ°дәҶдёӨж¬ЎпјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒеҲ©з”Ёе…¶д»–зҡ„дёҖдәӣйў„еӨ„зҗҶе‘Ҫд»ӨжқҘи§„йҒҝиҝҷз§Қжғ…еҶөпјҢзңӢзңӢдёӢйқўзҡ„д»Јз Ғпјҡ

жҲ‘д»¬еҶҚжқҘзңӢзңӢеўһеҠ дәҶиҝҷдәӣйў„еӨ„зҗҶе‘Ҫд»Өд№ӢеҗҺзҡ„йў„еӨ„зҗҶж–Үд»¶еҶ…е®№пјҡ

OKпјҢиҝҷе°ұжӯЈеёёдәҶгҖӮеҰӮжһңжҲ‘д»¬еңЁ A.h дёӯеҢ…еҗ« B.hпјҢз„¶еҗҺеҸҲеңЁ B.h дёӯеҢ…еҗ« A.hпјҢе…·дҪ“д»Јз ҒеҰӮдёӢпјҡ

жҲ‘д»¬еҶҚжқҘзңӢзңӢз»ҸиҝҮ gcc -E B.h еӨ„зҗҶд№ӢеҗҺзҡ„ж–Үд»¶еҶ…е®№пјҡ

жҲ‘д»¬еҸ‘зҺ° A.h е’Ң B.h йҮҚеӨҚеҮәзҺ°пјҢиҝҷжҳҜеӣ дёәиҝҷдёӘж—¶еҖҷ A.h е’Ң B.h ж–Үд»¶дә’зӣёеј•з”ЁеҜјиҮҙзҡ„гҖӮд»ҺзҗҶи®әдёҠжқҘи®ІпјҢиҝҷдёӘж—¶еҖҷдјҡж— йҷҗеҫӘзҺҜдёӢеҺ»пјҢзӣҙиҮідё–з•Ңз»Ҳз»“гҖӮеңЁиҝҷйҮҢжңҖеҗҺдјҡеҮәзҺ°дёҖеҸҘ 1 error generated.зҡ„жҸҗзӨәжҳҜ gcc ејәиЎҢдёӯж–ӯдәҶиҝҷдёӘйў„еӨ„зҗҶзҡ„иҝҮзЁӢпјҢжүҖд»ҘжҲ‘д»¬жүҚиғҪзңӢеҲ°иҝҷж ·зҡ„з»“жһңгҖӮйӮЈжҲ‘д»¬еҸҜд»ҘжҖҺд№ҲеҒҡпјҹеҪ“з„¶жҳҜеҲ©з”ЁеүҚйқўиҜҙзҡ„йў„еӨ„зҗҶе‘Ҫд»ӨжқҘйҒҝе…ҚеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳгҖӮзңӢдёӢйқўзҡ„д»Јз Ғпјҡ

иҝҷдёӘж—¶еҖҷдҪҝз”Ё gcc -E B.h е°ұеҸҜд»ҘжӯЈеёёзҡ„иҝӣиЎҢйў„еӨ„зҗҶпјҢжңҖеҗҺзҡ„з»“жһңеҰӮдёӢпјҡ

жүҖд»ҘCзЁӢеәҸе‘ҳжҖ»жҳҜйңҖиҰҒйҖҡиҝҮеҗ„з§ҚжүӢж®өпјҲжҜ”еҰӮпјҡ#pragma onceпјүжқҘйҳІиҢғжӯӨзұ»дәӢд»¶зҡ„еҸ‘з”ҹгҖӮ

дәҢгҖҒ#import

жҲ‘д»¬еңЁж–Үд»¶дёӯйҖҡиҝҮ#importжқҘеҜје…Ҙ iAd Frameworkпјҡ

зј–иҜ‘жҠҘй”ҷпјҡ

йңҖиҰҒйҮҚж–°еҜје…Ҙе’Ңй“ҫжҺҘ Frameworkпјҡ

зј–иҜ‘жҲҗеҠҹпјҡ

д»ҺдёҠйқўзҡ„иҝҮзЁӢдёӯжҲ‘д»¬е°ұзҹҘйҒ“еңЁ Objective-C йЎ№зӣ®дёӯдҪҝз”Ё #import йңҖиҰҒжіЁж„ҸеҜје…Ҙе’Ңй“ҫжҺҘ FrameworkпјҢеҗҰеҲҷжҳҜдјҡжҠҘй”ҷзҡ„гҖӮ

йў„еӨ„зҗҶеҷЁеңЁзў°еҲ° #import е‘Ҫд»Өзҡ„ж—¶еҖҷпјҢе®ғдјҡйҮҮз”ЁйҖ’еҪ’зҡ„ж–№ејҸжҠҠиў«жүҖжңүеӨҙж–Үд»¶зҡ„еҶ…е®№еӨҚеҲ¶е№¶зІҳиҙҙеҲ°еҪ“еүҚж–Үд»¶дёӯпјҢеҰӮжһңж–Үд»¶дҫқиө–еұӮж¬ЎжҜ”иҫғж·ұе°ұдјҡйҖ жҲҗйў„еӨ„зҗҶеҗҺзҡ„ж–Үд»¶еҶ…е®№дҪ“з§ҜеӨ§е№…еәҰеҸҳеӨ§гҖӮ

жҜ”еҰӮеҜје…Ҙ UIKit зҡ„ж—¶еҖҷеҸӘйңҖиҰҒдёҖиЎҢд»Јз Ғпјҡ

йў„еӨ„зҗҶд№ӢеҗҺдјҡеҸҳжҲҗ200еӨҡиЎҢпјҲUIKit.h ж–Үд»¶жңү200еӨҡиЎҢд»Јз Ғпјүпјҡ

жҺҘдёӢжқҘиҝҳйңҖиҰҒйҖ’еҪ’зҡ„жҠҠжҜҸдёӘеӨҙж–Үд»¶зҡ„еҶ…е®№еұ•ејҖпјҢжңҖеҗҺзҡ„з»“жһңе°ұжҳҜдёҖиЎҢд»Јз ҒеҸҳжҲҗи¶…иҝҮ11000иЎҢд»Јз ҒгҖӮеҰӮжһңжңүеӨҡдёӘж–Үд»¶йғҪеҢ…еҗ«жқҘ UIKit зҡ„еӨҙж–Үд»¶пјҢиҝҷж ·е°ұдјҡи®©жҜҸдёӘж–Үд»¶зҡ„дҪ“з§ҜйғҪдјҡеҸҳеҫ—еҫҲеӨ§пјҢзј–иҜ‘иҝҮзЁӢд№ҹдјҡеҸҳеҫ—и¶ҠжқҘи¶Ҡж…ўгҖӮиҝҷз§ҚйҖ’еҪ’зҡ„ж–№ејҸдјҡи®©йЎ№зӣ®зҡ„зј–иҜ‘ж—¶й—ҙеҸҳжҲҗпјҡM source files + N headers => M x N compile timeгҖӮ

жүҖд»ҘиҝҷдёӘж—¶еҖҷжңүдёҖдёӘдјҳеҢ–ж–№жі•е°ұжҳҜжҠҠйЎ№зӣ®дёӯйў‘з№Ғиў«еј•з”Ёзҡ„ж–Үд»¶ж”ҫеҲ° PCHпјҲPre-Compile Headerпјүж–Үд»¶дёӯгҖӮPCH дјҡиў«зј–иҜ‘дёҖж¬Ўе№¶дё”дјҡиў«зј“еӯҳпјҢиҝҷе°ұеҸҜд»Ҙзј©зҹӯзј–иҜ‘ж—¶й—ҙпјҢжҲ‘д»¬д№ҹдёҚйңҖиҰҒеңЁдёҚеҗҢзҡ„ж–Үд»¶йҮҢйқўж·»еҠ importиҜӯжі•гҖӮ

еҪ“з„¶пјҢPCH д№ҹжңүиҮӘе·ұзҡ„зјәзӮ№пјҡ

  • з»ҙжҠӨиҙҹжӢ…пјҡйҡҸзқҖйЎ№зӣ®еҸҳеҫ—и¶ҠжқҘи¶ҠеӨҚжқӮпјҢжҲ‘д»¬е°ұдјҡдёҚеҒңзҡ„еҫҖPCHж–Үд»¶еҠ е…ҘеҶ…е®№пјҢеҶ…е®№дёҖж—ҰеҸҳеӨҡе°ұдјҡеҸҳеҫ—дёҚеҘҪз»ҙжҠӨгҖӮпјҲиҝҷд№ҹжҳҜжҲ‘д»¬е№іеёёеңЁйЎ№зӣ®дёӯиҰҒйҒҝе…ҚеңЁ ViewController еҒҡеӨӘеӨҡдәӢжғ…зҡ„пјҢиҰҒз ”з©¶ MVVMзҡ„зјҳж•…гҖӮпјү

  • е‘ҪеҗҚз©әй—ҙжұЎжҹ“

жңҖеҗҺпјҢз»ҷеӨ§е®¶жҸҗдҫӣдёҖдёӘдҫӢеӯҗзңӢзңӢ #import зј–иҜ‘еҮәжқҘд№ӢеҗҺзҡ„ж–Үд»¶еҶ…е®№пјҡ

дҪҝз”Ё gcc -E B.h иҝӣиЎҢйў„еӨ„зҗҶд№ӢеҗҺзҡ„еҶ…е®№еҰӮдёӢпјҡ

жҲ‘д»¬еңЁB.hдёӯжңүдёӨдёӘ #import вҖңA.hвҖқпјҢдҪҶжҳҜиҝҷдәӣеҶ…е®№и·ҹжҲ‘д»¬д№ӢеүҚеңЁ A.h е’Ң B.h ж–Үд»¶дёӯдҪҝз”Ё #include е’Ңе…¶д»–йў„еӨ„зҗҶе‘Ҫд»Өд№ӢеҗҺзҡ„еӨ„зҗҶз»“жһңеҫҲзӣёдјјпјҢжүҖд»ҘжҲ‘д»¬е°ұжҳҺзҷҪдәҶ #import еӨ§жҰӮеҒҡдәҶд»Җд№ҲдәӢгҖӮ

дёүгҖҒ@import

еңЁ2012е№ҙзҡ„ LLVM еӨ§дјҡдёҠпјҢиӢ№жһңзҡ„ Doug Gregor йҰ–ж¬ЎжҸҗеҮәдәҶ Objective-C дёӯзҡ„ ModuleгҖӮдҪҝз”Ё @import ж–№ејҸеҜје…ҘжңүеҮ дёӘеҘҪеӨ„пјҡ

  • дёҚйңҖиҰҒеғҸ #import дёҖж ·еҫ—жүӢеҠЁеҺ»й“ҫжҺҘ FrameworkпјҢ@importдјҡиҮӘеҠЁеҺ»й“ҫжҺҘ

  • @import е·ҘдҪңж–№ејҸе’Ң PCH еҫҲеғҸпјҢдҪҶжҳҜ @import иҰҒжҜ” PCH зҡ„ж•ҲзҺҮй«ҳеҮәи®ёеӨҡ

  • @import еҜје…Ҙ Modul дјҳеҢ–ж–Үд»¶дҪ“з§ҜеҸҳеӨ§гҖҒзј–иҜ‘йҖҹеәҰеҸҳж…ўзҡ„й—®йўҳ

  • еҸҜд»ҘйғЁеҲҶеҜје…ҘпјҲ@import Framework.AпјүжҲ–е…ЁйғЁеҜје…ҘпјҲ@import Frameworkпјү

жүҖд»ҘпјҢе»әи®®еӨ§е®¶е°ҪйҮҸдҪҝз”Ё @import жқҘеҜје…Ҙж–Үд»¶гҖӮеҰӮжһңдҪ д»ҘеүҚзҡ„йЎ№зӣ®з”Ёзҡ„жҳҜ #importпјҢйӮЈд№ҲдҪ д№ҹдёҚйңҖиҰҒжӢ…еҝғпјҢжҲ‘д»¬еҸӘйҖҡиҝҮ Build Settings ејҖеҗҜ Modules йҖүйЎ№пјҲзңӢдёӢеӣҫпјүпјҢ#import е’Ң #include дјҡиҮӘеҠЁиў«жҳ е°„жҲҗ @importпјҢжүҖд»ҘдҪ дёҚйңҖиҰҒжӣҙж”№еҺҹжқҘзҡ„д»Јз Ғд№ҹиғҪдә«еҸ— @importеёҰжқҘзҡ„еҘҪеӨ„гҖӮ

иҜҰз»ҶеҶ…е®№еҸҜд»ҘзңӢзңӢиӢ№жһң2013е№ҙзҡ„ Advances in Objective-CпјҢйҮҢйқўе°ұиҜҰз»Ҷд»Ӣз»ҚдәҶ ModuleгҖӮ

еӣӣгҖҒж–Үд»¶и·Ҝеҫ„

жҺҘдёӢжқҘжҲ‘д»¬жқҘдәҶи§ЈдёҖдёӢ #include е’Ң #include вҖңfileвҖқпјҡ

  • #include \: иЎЁзӨәзј–иҜ‘еҷЁдјҡзӣҙжҺҘеҲ°зі»з»ҹи®ҫе®ҡзҡ„зӣ®еҪ•дёӢеҜ»жүҫжҢҮе®ҡзҡ„ж–Үд»¶гҖӮ

  • #include вҖңfileвҖқ: иЎЁзӨәзј–иҜ‘еҷЁдјҡеҲ°еҪ“еүҚзҡ„зӣ®еҪ•дёӢеҜ»жүҫжҢҮе®ҡзҡ„ж–Үд»¶пјҢеҰӮжһңжүҫдёҚеҲ°пјҢеҲҷдјҡеҺ»зі»з»ҹи®ҫе®ҡзҡ„зӣ®еҪ•дёӢеҜ»жүҫжҢҮе®ҡзҡ„ж–Үд»¶гҖӮ
